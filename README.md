НАЗВА:Serenify
ТЕМА: Проєктування та розроблення інтерактивного додатку для відстеження психоемоційного стану людини 

АВТОР: Караков Кирил Сергійович
НАУКОВИЙ КЕРІВНИК: РАЙТЕР ОРЕСТ КОНСТЯНТИНОВИЧ

ОПИС: Розробленний інтерактивний додаток передбачає можливість запису користувачем своїх емоційних станів та спілкування з інтегрованим чат ботом. Також в частину записів своїх емоційних станів інтегровано класифікатор емоцій для оцінки повідомлень користувача.

Для запуску потрібно( всі команди виконуються в терміналі):

1. Клонувати репозиторій
2. Перейти до папки з RestApi
3. Встановити бібліотеку poetry: brew install poetry
4. Встановити всі бібліотеки: poetry install
5. Увійти у poetry shell : poetry shell
6. Після встановлення бібліотек запустити сервер : poetry run uvicorn server:app --port 8000
7. Повернутись до корневої папки
8. Перейти до папки frontend
9. Встановити потрібні пакети: npm install
10. Запустити проект: npm run dev
